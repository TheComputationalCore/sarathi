# Code of Conduct

## Our Pledge

We as contributors and maintainers pledge to make participation in the Sarathi project a harassment-free experience for everyone.

We are committed to fostering an open and welcoming community.

---

## Our Standards

Examples of behavior that contributes to a positive environment:

* showing respect for differing viewpoints
* providing constructive feedback
* being welcoming to new contributors
* focusing on improving the project

Examples of unacceptable behavior:

* harassment or discrimination
* personal attacks
* trolling or insulting comments
* publishing private information without permission

---

## Enforcement

Project maintainers are responsible for clarifying and enforcing the standards of acceptable behavior.

They may remove or edit contributions that violate this Code of Conduct.

---

## Scope

This Code of Conduct applies within:

* project repositories
* issue trackers
* pull requests
* community discussions

---

## Reporting Issues

If you experience or witness unacceptable behavior, please report it by opening a GitHub issue.

Project maintainers will review and respond appropriately.
