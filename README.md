# Sarathi – Civilizational Mobility Platform

A full-stack intelligent mobility and heritage exploration platform.

## Tech Stack
- Spring Boot
- PostgreSQL
- Redis
- React + Tailwind
- Docker
- Razorpay
- WebSockets

## Features
- Secure JWT Authentication
- Seat Locking with Redis
- Real-time Booking Updates
- Payment Gateway Integration
- Civilizational Trail Engine
- Admin Dashboard
- Production-ready Dockerized Setup

## Deployment
- Backend: Render
- Database: NeonDB
- Frontend: Docker + Caddy
